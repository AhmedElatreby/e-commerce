E-Commerce 
===
Abstract: A REST API for online e-commerce system




## Usage
- Install Dependencies
  ```
  npm install
  ```
- Run App
  ```
  npm start
  ```
## Technology Stack
- Node js
- Express Js
- PostgreSQL
- Redis
